# Tinder-like Card Swipe in Flutter

A Flutter project demonstrating a Tinder-like card swipe UI using Provider for state management.

## Folder Structure


lib/
  main.dart
  screens/
    home_screen.dart
  models/
    profile.dart
  services/
    profile_service.dart


## Dependencies

- Flutter
- cupertino_icons
- provider

## Getting Started

1.  Clone the repository.
2.  Run `flutter pub get`.
3.  Run `flutter run`.
